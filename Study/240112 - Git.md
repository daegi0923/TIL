# #240112 - Git

## Git

- **git reset** : stage에 있는파일 unstaging
    - **git reset <파일명>** : 파일하나만
    - **git reset**  : 모든파일 언스테이징
- **git add .** : 모든 파일 스테이징
- **git commit — amend** : 커밋 메시지 바꾸기
    1. i 키 눌러서 insert로 바꿔야함
    2. insert 끄고 :wq 입력하면 저장하고 나가기 됨
